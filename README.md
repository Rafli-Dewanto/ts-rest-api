# ts-rest-api


Rest API implementation using typescript


## Start Dev

To run development, run the following command

```bash
  npm run dev
```


## Run Locally

Clone the project

```bash
  git clone git@github.com:Rafli-Dewanto/ts-rest-api.git
```

Go to the project directory

```bash
  cd ts-rest-api
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
